# Добро пожаловать

**MTS AI Assistant** — это мощный бэкенд для ИИ-ассистента, построенный на базе современных технологий: **FastAPI**, **LangGraph** и **Qdrant**.

Этот проект спроектирован для использования в качестве "умного мозга" в связке с OpenWebUI или любым другим frontend-клиентом, поддерживающим стандартный OpenAI API протокол.

## Ключевые возможности

- **Эксклюзивно на MWS API**: Глубокая интеграция с моделями MWS: 
    - `llama-3.3-70b-instruct` (главная логика / Deep Research)
    - `qwen2.5-72b-instruct` / `llama-3.1-8b` (быстрый чат и классификация)
    - `qwen3-coder-480b-a35b` (специализированная модель для кода и анализа данных)
    - `qwen3-embedding-8b` (продвинутые эмбеддинги для памяти)
    - `qwen-image-lightning` (мгновенная генерация изображений)
    - `whisper-turbo-local` (ASR / транскрипция речи)
    - `qwen2.5-vl-72b` (Vision / глубокое зрение)
- **Умный роутинг (Smart Router)**: Автоматическое определение намерений с Fast-Track механизмом. Поддерживает: `chat`, `search`, `research`, `image`, `presentation`, `audio`, `data` (аналитика), `website` (лендинги).
- **Data Analyst Agent**: Встроенная песочница Python для проведения EDA (разведочного анализа), построения графиков и сложных вычислений по текстовым данным.
- **Website & Doc Generator**: Генерация полноценных сайтов одним запросом, а также создание структурированных документов в форматах Word (DOCX) и PowerPoint (PPTX).
- **LangGraph Orchestration**: Использование графов состояний для проведения Deep Research и многоэтапного выполнения задач.
- **Долгосрочная память (Memory 2.0)**: Извлечение фактов и семантический поиск по истории диалогов через векторную базу Qdrant.
- **Интеграция с MCP**: Отдельный Model Context Protocol сервер, запускающийся как подпроцесс для работы с веб-поиском и парсингом страниц.
- **Универсальные API Endpoint-ы**: Поддержка текстового чата, потоковой генерации презентаций (PPTX) и транскрипции аудио.
