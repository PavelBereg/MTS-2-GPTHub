# Инструменты и Агенты (Tools)

Приложение реализует паттерн *Function Calling* и использует **Model Context Protocol (MCP)** для предоставления агенту (LangGraph) доступа к внешнему миру. 

Инструменты дают возможность языковой модели выходить за рамки своих тренировочных данных.

## Нативные Инструменты (Python)

Нативные интерфейсы расположены в `app/graph/tools.py`.

### Генерация Изображений
* **`generate_image_tool`**
  Используется напрямую агентом, либо вызывается Роутером в режиме `AUTO`, когда пользователь просит "нарисуй", "сгенерируй". Отправляет запрос к MWS API (`images/generations` модель `qwen-image-lightning`) и возвращает markdown-строку с встроенным URI изображения или ссылкой.

### Аналитика и Песочница Python
* **`execute_python`**
  Мощный инструмент для выполнения Python-кода в изолированной среде. Служит основой для режима **Data Analyst**.
  
  **Ключевые правила работы:**
  - **STOP TALKING, START CODING**: Агент обязан писать код при любом аналитическом запросе (EDA, графики).
  - **Контекстные данные**: Так как прямого доступа к файловой системе у агента нет, данные из чата хардкодятся в скрипт и читаются через `io.StringIO()`.
  - **Визуализация**: Все графики (matplotlib, seaborn) сохраняются в `static/downloads/` и выводятся пользователю в виде абсолютных Markdown-ссылок (`http://localhost:8000/...`).
  
### Генератор Веб-сайтов
* **`website_generation`**
  Специализированный инструмент для построения современных адаптивных сайтов. Генерирует структуру проекта (HTML, CSS, JS), создает тематические изображения через MWS Vision/Image API и упаковывает результат в готовый ZIP-архив.

## Инструменты Сервера MCP (Model Context Protocol)

Сервер `mcp_server/server.py` работает как отдельный процесс, общаясь с основным приложением через стандартный I/O. Этот подход гарантирует безопасность выполнения и распределение ответственности.

### Веб Поиск (`search_web`)
Использует библиотеку `duckduckgo_search` для поиска информации в интернете в реальном времени. Во избежание галлюцинаций инструмент проверяет доступность ссылки (`HEAD` запросом), отфильтровывает нерабочие сайты и "трекерные" URL DuckDuckGo. Имеет логику фильтрации новостей за последний год, чтобы агент получал самую свежую и актуальную информацию.

### Скрейпинг страниц (`scrape_url`)
Используется, когда пользователь отправляет ссылку на страницу и просит её "кратко пересказать" или проанализировать. Инструмент выполняет GET запрос (с подменным Modern User-Agent), и с помощью `BeautifulSoup` очищает страницу от рекламы, скриптов, стилей, навигации. В агент возвращается только чистый, смысловой текст страницы со строгим ограничением по длине для экономии токенов контекстного окна.
