# Спецификация функциональных возможностей (Features Status)

В данном документе представлен актуальный статус реализации функциональных модулей проекта MTS AI Assistant.

| Название функции | Статус | Описание | Технические детали |
|:---|:---|:---|:---|
| Текстовый и голосовой чат | Implemented | Поддержка двустороннего общения. | Модели: llama-3.3-70b / qwen2.5-72b. |
| Аналитика данных (EDA) | Implemented | Разведочный анализ, расчеты и визуализация через Python. | Модель: qwen3-coder-480b. Инструмент: execute_python. |
| Генератор Сайтов | Implemented | Создание лендингов и веб-интерфейсов "под ключ" (ZIP). | Адаптивная верстка, MWS Images для контента. |
| Генерация презентаций | Implemented | Создание файлов в формате PPTX по запросу. | Модуль: PPTX Helper. Библиотека: python-pptx. |
| Генерация документов | Implemented | Создание структурированных файлов в формате Word (DOCX). | Модуль: DOCX Helper. |
| Распознавание речи (ASR) | Implemented | Транскрипция аудиосообщений в текст. | Модель: whisper-turbo-local. |
| Визуальный анализ (VLM) | Implemented | Анализ содержимого загруженных изображений. | Модель: qwen2.5-vl-72b. |
| Генерация изображений | Implemented | Создание графики по описанию. | Модель: qwen-image-lightning. |
| Веб-поиск | Implemented | Поиск актуальной информации в реальном времени. | DuckDuckGo (через MCP/DDGS). |
| Веб-скрейпинг | Implemented | Извлечение очищенного текста с веб-страниц. | Библиотека: BeautifulSoup4. |
| Долгосрочная память | Implemented | Хранение фактов и истории через векторную БД. | Qdrant Vector DB. Эмбеддинги: qwen3-embedding-8b. |
| Глубокое исследование | Implemented | Многошаговый поиск и компиляция сложных отчетов. | LangGraph Deep Research Agent. |
| Файловое взаимодействие | Implemented | Работа с данными через Python Sandbox и генерация DOCX/PPTX. | Изолированная среда выполнения. |
